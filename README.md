# ToDoList App

A simple command-line ToDo List application built with Python.

## Features

* Add tasks
* Delete tasks by ID
* Search tasks by ID
* Search tasks by name
* Prevent duplicate task names
* Sort tasks by priority
* Auto-save tasks
* Load tasks automatically on startup
* JSON data storage
* Object-Oriented Programming (OOP)

## Technologies Used

* Python
* JSON
* Git
* GitHub

## Project Structure

* `todo_app.py` : Main application file
* `ToDoList.json` : Stores task data

## How to Run

1. Clone the repository

```bash
git clone https://github.com/emataka/ToDoList.git
```

2. Navigate to the project folder

```bash
cd ToDoList
```

3. Run the application

```bash
python todo_app.py
```
